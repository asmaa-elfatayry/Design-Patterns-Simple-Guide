Null Object Pattern is a **behavioral design pattern** that provides a default object with "do nothing" behavior instead of returning `null`.

---

## âš  Important Note

> Null Object **protects the absence of the object itself**, not the values you pass to the object.  
> Example: if you pass `null` as an argument to a method of the object, the Null Object **does not handle it automatically**.  
> You must handle `null` values inside the method if you want full safety.
